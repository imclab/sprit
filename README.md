Sprit
=====

Computer vision approach to keep track of your fuel burning activities.

Pipeline
 1. Localize candidates for text/numbers in given image
 2. Recognize identfied candidates i.e. perform OCR (how?) 
 3. High level reasoning: classify which digits represent total fuel, total money, money/fuel, etc.
 4. Plot some stats, get current date from image... 

Interesting papers
 - [EpshteinCVPR10] Detecting text in natural scenes using SWT
 - [NeumannCVPR12] Real-time scene text localisation and recognition
